## 🚨 严重问题（必须修复）

### 1. Session 管理问题
**位置**: `src/app/api/auth/login/route.ts`  
**问题**: 同一用户多次登录会创建多个 Session，导致 Session 表无限增长  
**影响**: 数据库性能下降，存储空间浪费  
**修复**: 登录时删除该用户的所有旧 Session

### 2. 中间件 Session 验证缺失
**位置**: `src/middleware.ts`  
**问题**: 只验证 JWT token，不验证 Session 是否存在于数据库  
**影响**: 即使用户被登出（Session 删除），token 有效期内仍可访问  
**修复**: 添加 `validateSession` 调用

### 3. 无限点赞/Fork
**位置**: `src/app/api/projects/[id]/action/route.ts`  
**问题**: 没有防止用户对同一项目重复点赞或 Fork  
**影响**: 数据污染，统计数据不准确  
**修复**: 创建 Like 表记录用户点赞历史

### 4. Fork 类型不匹配
**位置**: `src/app/api/projects/[id]/action/route.ts`  
**问题**: 直接赋值 `tags: project.tags`，但 schema 中 tags 是 String（JSON）  
**影响**: 运行时错误  
**修复**: 使用 `JSON.stringify(project.tags)`

### 5. JSON 解析无异常处理
**位置**: `src/app/api/chat/route.ts`  
**问题**: `JSON.parse(conversation.messages)` 可能抛出异常  
**影响**: 整个请求失败  
**修复**: 添加 try-catch

### 6. 消息无限制增长
**位置**: `src/app/api/chat/route.ts`  
**问题**: 对话消息无数量限制，messages 字段可能无限增长  
**影响**: 数据库性能下降，查询变慢  
**修复**: 限制最大消息数，裁剪旧消息

### 7. JWT_SECRET 默认值不安全
**位置**: `src/lib/auth.ts`  
**问题**: JWT_SECRET 有默认值 `your-secret-key-change-in-production`  
**影响**: 生产环境如果忘记配置，使用不安全的密钥  
**修复**: 移除默认值，强制要求配置

### 8. 缺少 API 限流
**全局问题**  
**问题**: 所有 API 端点都没有限流保护  
**影响**: 
- 暴力破解攻击
- API 滥用
- 数据库垃圾数据
- DDoS 攻击  
**修复**: 添加 rate limiting

### 9. 文件路径遍历漏洞
**位置**: `src/app/api/upload/route.ts`  
**问题**: 没有验证文件名，可能包含恶意路径  
**影响**: 路径遍历攻击  
**修复**: 验证文件名，移除路径分隔符

### 10. 无限文件上传
**位置**: `src/app/api/upload/route.ts`  
**问题**: 没有限制用户总存储空间  
**影响**: 磁盘空间耗尽  
**修复**: 记录用户已用空间，限制总量

### 11. 文件不清理
**位置**: 全局  
**问题**: 删除 FileUpload 记录时，不删除实际文件  
**影响**: 磁盘空间浪费  
**修复**: 实现 cascade delete 或定期清理

## ⚠️ 中等问题（建议修复）

**位置**: `src/app/profile/page.tsx`  
**问题**: `useEffect(() => { loadUserData() }, [])` 应该有依赖  
**影响**: 可能导致无限循环或数据不更新  
**修复**: 添加正确的依赖数组

**问题**: 大量使用 `any` 类型  
**影响**: 失去类型安全优势

**影响**: 查询性能差  
**修复**: 添加必要的索引

### 15. .env.local 提交到代码库
**位置**: 项目根目录  
**问题**: `.env.local` 应该被忽略  
**影响**: 敏感信息可能泄露

### 17. Zustand 状态不同步

### 19. CORS 配置缺失

**问题**: Next.js API 没有配置 CORS
